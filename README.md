# 🐈 해결캣 (HaegyeolCat) – Front-End

> 고양이를 키우며, 서로의 불안을 해결하는 공감 기반 커뮤니티 서비스

---

## 📌 프로젝트 개요

**해결캣**은 사용자의 일상 속 불안을  
**공감 · 조언 · 기여**의 과정을 통해 해소하도록 돕는 모바일 서비스입니다.

사용자는 자신의 불안을 글로 공유할 수 있고,  
다른 사용자는 댓글을 통해 공감하거나 해결책을 제시합니다.  

이 과정에서 고양이를 돌보고 성장시키는 **감정 회복 메타포**를 통해  
불안 → 공감 → 기여 → 회복 → 성장의 선순환 구조를 제공합니다.

본 레포지토리는 **해결캣 서비스의 Front-End 구현을 담당**합니다.

---

## 🧩 문제 정의 (Problem)

다음과 같은 순간의 불안은 명확한 해결책이 없기에 더 깊어집니다.

- 특별한 이유 없이 문득 불안해지는 순간
- 주변 사람들과 자신을 비교하게 될 때
- 취업 준비 과정에서 반복되는 좌절
- 프로젝트·해커톤 마감 직전의 압박
- 관계 속에서의 불안과 단절
- 현실적인 생존 문제에서 오는 불안

> 해결캣은 **“불안을 혼자 견디지 않아도 되는 공간”**을 목표로 합니다.

---

## 💡 해결 방안 (Solution)

### 불안 → 공감 → 기여 → 회복 → 성장

- 사용자는 **불안한 고양이**의 형태로 자신의 상태를 표현
- 타인의 불안에 공감하고 해결책을 제시
- 기여 활동을 통해 **포인트(캣잎)** 획득
- 고양이의 상태가 점차 회복 및 성장
- 결국 **내 불안을 해결받을 수 있는 구조** 형성

---

## 🚀 주요 기능

### ✍️ 불안 게시글 작성
- 사용자가 자신의 불안을 게시글로 작성
- 불안 카테고리 기반 정리

### 💬 공감 & 해결 댓글
- 타인의 불안에 공감 댓글 작성
- 구체적인 해결책 제안 가능

### ✅ 해결책 채택
- 게시글 작성자가 가장 도움이 된 댓글을 해결책으로 채택
- 채택 시 기여 점수 상승

### 🐾 고양이 성장 시스템
- 활동에 따라 캣잎 포인트 획득
  - 게시글 작성
  - 댓글 작성
  - 해결책 채택
  - 공감 받기 / 공감하기
- 포인트에 따라 고양이 상태 변화
  - 매우 불안 → 어느 정도 불안 → 그냥 그래 → 조금 행복 → 매우 행복

## 🎯 가치 제안 (Value Proposition)

- 타인의 불안을 해결하면 **보상이 주어지는 구조**
- 공감이 일회성 위로로 끝나지 않음
- 기여가 시각적 성장(고양이)으로 명확히 보임
- “내가 누군가에게 도움이 됐다”는 성취감 제공
